# api-contract-designer

Skill para diseñar contratos REST profesionales antes de implementar backend.

## Qué produce

- Endpoints estructurados
- Requests y responses definidos
- Códigos HTTP correctos
- Manejo profesional de errores
- Seguridad base

## Ejemplos de uso

- "Diseñame la API para un sistema de tareas con autenticación."
- "Necesito definir endpoints para un e-commerce."
- "Cómo debería estructurar la API de usuarios y roles."

## Output esperado

Contrato listo para implementar en Express o cualquier backend REST.
