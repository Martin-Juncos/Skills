# Quick Checklists

CORS:
- origin correcto
- credentials si cookies
- preflight OPTIONS

Auth:
- expiración token
- clock skew
- claims esperados
- middleware ordenado

DB:
- connection string
- pool size/timeouts
- migraciones aplicadas
- índices

CI:
- node version
- lockfile consistente
- env vars definidas
- scripts correctos
