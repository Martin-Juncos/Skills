# systematic-debugging-framework

Skill para depurar problemas fullstack con un proceso profesional.

## Útil para

- Bugs de React
- Errores de Express
- Problemas de DB
- CORS y auth
- Fallas de CI/build
- Deploy roto
- Comportamientos intermitentes

## Ejemplos

- "Mi CI falla con error de lint, ayudame a resolverlo."
- "CORS me bloquea en producción, qué reviso."
- "El login funciona local pero falla en deploy."
