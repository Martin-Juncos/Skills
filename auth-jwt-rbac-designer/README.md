# auth-jwt-rbac-designer

Skill para diseñar autenticación JWT y autorización por roles (RBAC) de forma profesional.

## Qué produce

- Flujos de login/refresh/logout
- Estrategia access token + refresh token
- Roles y permisos
- Middlewares y protección de rutas
- Integración frontend-backend
- Checklist de seguridad

## Ejemplos

- "Necesito login con JWT y roles admin/user."
- "Quiero refresh tokens y rutas protegidas."
- "Diseñame RBAC para productos, pedidos y usuarios."

## Resultado

Diseño listo para implementar con Express y React.
