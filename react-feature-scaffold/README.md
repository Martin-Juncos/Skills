# react-feature-scaffold

Skill para generar features organizadas en React con arquitectura profesional.

## Qué produce

- Estructura por feature
- Componentes organizados
- Hook personalizado
- Servicio de integración API
- Manejo de estados (loading, error)

## Ejemplos

- "Generame la feature products con listado y creación."
- "Necesito integrar users con API usando Redux."
- "Creá la feature orders con autenticación."

## Resultado

Feature lista para integrarse al proyecto.
