# express-api-scaffold

Skill para generar estructura profesional de backend en Express.

## Qué produce

- Estructura por capas
- Código base organizado
- Validación de entrada
- Manejo de errores
- Integración opcional de autenticación

## Ejemplos

- "Generame el backend para el recurso products con CRUD."
- "Necesito el módulo users con JWT."
- "Creá el endpoint orders con Sequelize."

## Resultado

Backend listo para integrar en el proyecto.
