# C4 Context (opcional, textual o Mermaid)

Cuándo incluirlo:
- Cuando el usuario pida “diagrama”, “C4”, “documentación ejecutiva”, “contexto del sistema”.
- Cuando haya muchas integraciones externas.

Contenido mínimo:
- Descripción corta del sistema (1 línea)
- Personas/actores (humanos y sistemas)
- Sistemas externos y dependencias
- Principales flujos (alto nivel)

Mermaid (opcional)
Usar C4Context si el entorno lo soporta. Caso contrario, entregar versión textual.
