# db-migration-and-index-strategy

Skill para definir migraciones e índices de forma profesional en Postgres (Sequelize) y/o Mongo (Mongoose).

## Qué produce

- Estrategia de migraciones
- Convenciones y flujo de trabajo
- Índices recomendados por patrón de consulta
- Seguridad para despliegues (rollout/rollback)
- Checklist pre/post deploy

## Ejemplos

- "Estoy con Postgres + Sequelize, definime estrategia de migraciones e índices."
- "Tengo queries lentas en productos con filtros y paginado, qué índices uso."
- "Uso Mongo con Mongoose, cómo manejo cambios de esquema y performance."
