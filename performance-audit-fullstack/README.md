# performance-audit-fullstack

Skill para auditar performance end-to-end en proyectos fullstack.

## Qué produce

- Plan de medición
- Diagnóstico FE/BE/DB
- Hipótesis con pruebas
- Quick wins
- Plan de optimización por fases

## Ejemplos

- "Mi API tarda 2–3s en /products con filtros, ayudame a optimizar."
- "La UI se siente pesada al navegar, qué reviso en React."
- "Tengo timeouts en producción, necesito auditoría completa."
