# documentation-and-onboarding

Skill para estructurar documentación profesional y procesos de onboarding.

## Qué produce

- README profesional
- Sistema de ADR
- Guía de contribución
- Guía de onboarding técnico
- Checklist documental

## Ejemplos

- "Necesito profesionalizar mi README."
- "Quiero agregar ADR a mi proyecto."
- "Voy a trabajar en equipo, armame documentación base."
