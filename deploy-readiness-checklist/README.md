# deploy-readiness-checklist

Skill para preparar un proyecto fullstack para producción con una checklist profesional.

## Qué produce

- Checklist priorizada (P0/P1/P2)
- Variables de entorno recomendadas
- Validaciones de build y runtime
- Seguridad mínima
- Observabilidad básica
- Rollback simple

## Ejemplos

- "Voy a desplegar React + Express + Postgres, ¿qué tengo que revisar?"
- "Mi deploy falla por variables de entorno, armame checklist."
- "Quiero checklist de producción para mi proyecto fullstack."
