# Clasificación de proyecto

Tipos comunes:

1. CRUD simple
2. E-commerce
3. SaaS multiusuario
4. Dashboard analítico
5. Landing + API
6. API-only
7. Producto educativo
8. Sistema interno empresarial

Cada tipo altera:
- Necesidad de auth
- Nivel de seguridad
- Complejidad de datos
- Observabilidad requerida
