# git-workflow-selector

Skill para definir el workflow profesional de Git y GitHub.

## Qué produce

- Estrategia de ramas
- Convención de commits
- Política de PR
- CI mínimo
- Versionado
- Checklist operativo

## Ejemplos de uso

- "Necesito un workflow profesional para mi proyecto React + Node."
- "Cómo debería organizar ramas en un equipo de 3 desarrolladores."
- "Quiero profesionalizar mi repositorio en GitHub."

## Resultado

Workflow listo para aplicar en cualquier proyecto fullstack.
